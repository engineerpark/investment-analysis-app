🚀 투자 포트폴리오 앱 테스트 플로우

✅ 완료된 기능 모두 완료

1. 기본 포트폴리오 자동 선택 NEW

  1. 홈 화면에서 "투자 설문 시작" 클릭
  2. 투자 성향 설문 완료 (보수적/균형/성장/초고위험)
  3. 포트폴리오 추천 화면 진입
  4. 성향에 맞는 기본 자산이 자동으로 선택됨 확인
  5. 추가/제거 가능 여부 테스트

2. 백테스팅 전용 화면 NEW

  1. 포트폴리오 대시보드에서 "백테스팅" 버튼 클릭
  2. 시계열 그래프 표시 확인
  3. 1년/3년/5년/10년 기간 버튼 테스트
  4. 리밸런싱 마커(파란점) 표시 확인
  5. 수익률, 샤프비율 등 지표 확인

3. AI 미래예측 화면 NEW

  1. 포트폴리오 대시보드에서 "미래예측" 버튼 클릭
  2. 3년 학습 데이터(실선) 표시 확인
  3. 6개월/1년 예측 데이터(점선) 표시 확인
  4. 예측 신뢰도, 모델 정확도 확인
  5. 경제 지표 반영 정보 확인

4. UX/UI 색상 개선 NEW

  1. 각 화면의 버튼 색상 확인
  2. Primary: 주요 액션 (파란색)
  3. Success: 긍정적 액션 (초록색)
  4. Danger: 위험한 액션 (빨간색)
  5. Warning: 주의 액션 (주황색)
  6. Info: 정보성 액션 (하늘색)
